# Codes send in server Responses

## Notes
* -100 --> get all notes
* -1 --> new note

## Status
* 200 --> OK
* 201 --> note saved
* 202 --> note created
* 203 --> note deleted
* 204 --> note updated

## Error
* 400 --> Server Error
* 401 --> note could not be saved
* 402 --> note could not be deleted
* 403 --> note could not be created
* 404 --> note not found
* 405 --> no note file found
* 406 --> note file write or read error

## Several
* 900 --> test successfull
